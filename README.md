#  Netflix Dashboard Report

## 📌 Introduction
This project is an interactive Netflix Dashboard created using Power BI. It helps in analyzing Netflix content based on different factors such as country, release year, and directors.

---

## Objective
The objective of this project is to:
- Analyze Netflix dataset
- Understand content distribution across countries
- Identify trends based on release years
- Create an interactive dashboard

---

## Tools & Technologies Used
- Power BI
- Data Visualization
- Netflix Dataset (CSV file)

---

## 📊 Features of Dashboard
- Country-wise content analysis
- Release year trend visualization
- Treemap showing distribution by country and director
- Interactive visuals for better understanding

---

## 📈 Insights
- United States has the highest number of content
- India and UK also have significant contributions
- Content production has increased over the years
